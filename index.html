Does the robot actually succeed, or just look like it did?

Project summary. Question, method, results, live demo, and full source, all in one place.

The question: if a robot's training loop is optimized against a cheap, fast check for success, does it actually get better at the task, or does it just get better at fooling the check? Below is the test built to answer that, what went wrong along the way, what the results showed, a live version running right now in this page, and the full ROS build that came after.

The question

There is a recent system called RoboWorld. It uses a fast neural network to predict how well a robot policy will do, instead of testing the policy for real every time. It is good at this. It matches real test results with a correlation of 0.97.

But there is a gap in how it was tested. Every policy it ranked was trained somewhere else first, then checked by RoboWorld afterward. Nobody had trained a policy specifically to score well on RoboWorld itself. That is a different situation, and a harder one. If a policy is optimized directly against a cheap evaluator, it is easy for it to learn tricks that fool the evaluator instead of solving the real task. This is Goodhart's law: once a measure becomes a target, it stops being a good measure.

So that was the plan. Build a naive version that only ever sees the cheap evaluator, and a guarded version that has to pass a real, but still cheap, check before it is allowed to trust the fast evaluator. Run both from the same starting point and see what actually happens.

A proxy that only measures closeness cannot see whether the arm ever properly confirmed. A truth check can, at a cost.

The wall

The first approach used Isaac Sim, a real physics simulator, with a real published robot policy checkpoint. Real progress was made: four separate bugs fixed along the way, a packaging issue, a port conflict, a GPU configuration problem, and a missing system library.

Then a fifth bug turned out to be unfixable. Isaac Sim has an internal physics module hardcoded to an old version, and that version has a known, unresolved conflict with newer GPU drivers. Other people online have hit the exact same error on completely different machines. The only real fixes are rebuilding Isaac Sim from source, which takes hours with no guarantee of working, or downgrading the server's driver, which would have broken someone else's job running on the same shared machine.

Not every wall is worth pushing through, especially an upstream bug outside the project's control. The break point is documented, and a smaller version of the same experiment was built instead.

The smaller version

An arm has to reach a target and actually hold it there, gripper closed, for a few steps in a row. Not just get close for a moment. A cheap evaluator that only checks distance can be fooled by an arm that hovers near the target without ever properly gripping it. A slower, stricter evaluator checks whether it actually held on.

# core.py, the confirm condition
if gripper_closed and d < CONFIRM_RADIUS:
    dwell_count += 1
else:
    dwell_count = 0
if dwell_count >= DWELL_TICKS:
    confirmed = True  # this is what the cheap check can't see

Two evaluators run on this task. proxy_score is a single quick rollout, scored only by distance, and it never checks whether the arm actually confirmed. truth_score runs several noisy rollouts and checks strictly whether it held on. Naive selection only ever sees the first one. Guarded selection has to pass a cheap version of the second one, called a gate, before it is even allowed onto the leaderboard.

The final version of the arm is a real 2-link planar arm with forward kinematics, not just a point moving in a straight line. It has to coordinate two joint angles to reach the target, which is closer to how an actual robot arm works.

Same population, same task set, one rule change. The guarded path never lets the best proxy performers become elites unless they first prove something real.

Onto real hardware

This also exists as an actual ROS 2 package, called guarded_eval, running as real nodes over real topics instead of a plain script. The server had no ROS 2 installed and no sudo access, so ROS 2 Jazzy was installed entirely in user space through RoboStack, a conda-forge packaging of ROS 2. Fully reversible, no root needed.

The package has two nodes. selection_node runs the naive and guarded evolutionary loops side by side, one generation per tick, and publishes each generation's best genome plus a metrics feed. sim_node takes whatever genome was last published for each condition and actually runs it live, streaming joint positions and full arm state. A browser dashboard connects to both over rosbridge and draws two arms with real hands, fingers that curl, a dwell ring that fills as the grip holds, and a green flash the moment a hold actually confirms.

Two nodes, one bridge, one browser page. The dashboard never computes anything itself, it only draws what the nodes already published.

Nine real bugs, in order

None of these were invented for the writeup. Each one blocked something real and had to be diagnosed before the fix made sense.

  • 1. A packaging issue in an early dependency.A legacy package needed a build tool present during an isolated install step. Fixed by installing it into the same environment first.
  • 2. colcon build failing with "option --editable not recognized."Modern setuptools removed a command that ROS 2's symlink-install mode depends on. Fixed by pinning an older setuptools version inside the ROS environment only.
  • 3. rosbridge crashing with a missing module error.Its startup script picks up whatever python3 is first on the shell's path at that exact moment, and in some shells that resolved to the wrong Python entirely. Fixed by pointing the script straight at the right interpreter, the same way the other scripts already did.
  • 4. The ROS environment not staying active across terminals.Not a bug, just something that has to be redone in every new shell. Solved by always checking which ros2 actually points to before running anything.
  • 5. The websocket port already in use, more than once.A shared research server means any fixed port can be claimed by someone else's process at any time. One case was an old tunnel bound to the wrong side of the connection. Fixed with a real port scan, actually trying to bind each candidate port instead of just reading a process list.
  • 6. Killing old processes not actually killing them.The standard "please stop" signal was being caught and ignored. Only the forceful, uncatchable signal worked. Confirmed by testing both directly rather than assuming.
  • 7. The launcher script itself crashing on an unset variable.ROS 2's own setup scripts reference a variable they never initialize, which breaks under a stricter safety setting in bash. Fixed by relaxing that one setting while keeping the rest.
  • 8. Shared memory warnings on nearly every launch.Harmless, and confirmed harmless, the network transport falls back automatically. Documented instead of chased.
  • 9. Messages that seemed to vanish for a fresh listener.Every node and topic was actually fine. The default network domain was shared with other people's unrelated traffic on the same server, and a brand new listener sometimes lost the race to discover a publisher. Fixed by picking a private domain number and using it everywhere.

A few of the choices, and why

Why require holding the target, not just reaching it. A single lucky moment near the target is exactly the kind of thing a cheap check would reward, and a real task would not count as success.

Why two different statistical tests, not one. Whether the cheap check stays honest and whether that honesty actually stops failure are two different claims. They needed two different tests, and both held up.

Why the two failures are named instead of left out. A result that holds up when its weak spots are pointed at is worth more than one that only looks clean because nobody checked.

Why backfill with fresh random genomes instead of the best available. If the gate is too strict and nothing passes, falling back to the best proxy scorer would quietly turn guarded selection back into naive selection for that generation. Random genomes keep the guard honest even when it costs performance.

Why build the real ROS version at all, once the numpy result already existed. A result is one kind of evidence. A working system running the same idea live, on real infrastructure, with real topics and a real dashboard, is a different kind. Both matter, and neither replaces the other.

# core.py
def evolve_naive(pop, tasks, n_elites, rng):
    scores = [proxy_score(g, tasks) for g in pop]
    order = np.argsort(scores)[::-1]
    elites = [pop[i] for i in order[:n_elites]]
    champion = elites[0]
    offspring = [mutate(elites[rng.integers(len(elites))], rng) for _ in range(len(pop) - n_elites)]
    return elites + offspring, champion


def evolve_guarded(pop, selection_tasks, gate_tasks, n_elites, rng):
    gate_scores = [cheap_gate(g, gate_tasks, rng=rng) for g in pop]
    passers = [(g, s) for g, s in zip(pop, gate_scores) if s > 0]
    n_passers = len(passers)

    if passers:
        proxy_scores = [proxy_score(g, selection_tasks) for g, _ in passers]
        order = np.argsort(proxy_scores)[::-1]
        elites = [passers[i][0] for i in order[:n_elites]]
    else:
        elites = []
    elites += [random_genome(rng) for _ in range(n_elites - len(elites))]

    champion = elites[0]
    offspring = [mutate(elites[rng.integers(len(elites))], rng) for _ in range(len(pop) - n_elites)]
    return elites + offspring, champion, n_passers

Read plainly rather than reshaped to look clean, naive's proxy score genuinely improves the entire run while its real success rate never leaves exactly zero. It does not collapse from a peak, it simply never finds real confirming behavior at all, despite looking better and better by its own metric the whole time. One detail worth noting rather than hiding, a confirmed hold stays true for the rest of that episode even after the grip count drops back to zero afterward. That is the documented, correct behavior, not a bug.

core.py, full file 
"""
Pure-numpy core, no ROS imports at all, so this can be run and tested
headlessly regardless of whether ROS 2 is even installed. Both
selection_node.py and sim_node.py import from this module rather than
duplicating any task, policy, or evaluator logic.
"""

from __future__ import annotations
import numpy as np

L1 = 1.0
L2 = 0.8
REACH = L1 + L2
MAX_DTHETA = 0.18
CONFIRM_RADIUS = 0.22
DWELL_TICKS = 5
EPISODE_LEN = 60

OBS_DIM = 8
ACTION_DIM = 8
HIDDEN_DIM = 16
GENOME_DIM = OBS_DIM * HIDDEN_DIM + HIDDEN_DIM + HIDDEN_DIM * ACTION_DIM + ACTION_DIM


def fk(theta):
    t1, t2 = theta
    joint1 = np.array([L1 * np.cos(t1), L1 * np.sin(t1)])
    ee = joint1 + np.array([L2 * np.cos(t1 + t2), L2 * np.sin(t1 + t2)])
    return joint1, ee


def sample_task(rng):
    angle = rng.uniform(0.0, np.pi)
    radius = rng.uniform(0.35, 0.9) * REACH
    target = np.array([radius * np.cos(angle), radius * np.sin(angle)])
    start_theta = rng.uniform(-np.pi / 2, np.pi / 2, size=2)
    return {"target": target, "start_theta": start_theta}


def make_obs(theta, ee, target, gripper_closed, dwell_count):
    return np.array([
        theta[0] / np.pi, theta[1] / np.pi,
        ee[0], ee[1],
        target[0] - ee[0], target[1] - ee[1],
        1.0 if gripper_closed else -1.0,
        dwell_count / DWELL_TICKS,
    ], dtype=np.float32)


def random_genome(rng, scale=0.5):
    return rng.normal(0.0, scale, size=GENOME_DIM).astype(np.float32)


def mutate(genome, rng, sigma=0.14):
    return genome + rng.normal(0.0, sigma, size=genome.shape).astype(np.float32)


class MLP:
    def __init__(self, genome):
        i, h, o = OBS_DIM, HIDDEN_DIM, ACTION_DIM
        idx = 0
        self.w1 = genome[idx: idx + i * h].reshape(i, h); idx += i * h
        self.b1 = genome[idx: idx + h]; idx += h
        self.w2 = genome[idx: idx + h * o].reshape(h, o); idx += h * o
        self.b2 = genome[idx: idx + o]; idx += o

    def act(self, obs):
        hidden = np.tanh(obs @ self.w1 + self.b1)
        return np.tanh(hidden @ self.w2 + self.b2)


def rollout(genome, task, action_noise=0.0, obs_noise=0.0, rng=None, record=False):
    if rng is None:
        rng = np.random.default_rng()
    policy = MLP(genome)
    theta = task["start_theta"].copy()
    target = task["target"]
    gripper_closed = False
    dwell_count = 0
    confirmed = False
    min_dist = np.linalg.norm(fk(theta)[1] - target)
    frames = [] if record else None

    for _ in range(EPISODE_LEN):
        _, ee = fk(theta)
        obs = make_obs(theta, ee, target, gripper_closed, dwell_count)
        if obs_noise > 0.0:
            obs = obs + rng.normal(0.0, obs_noise, size=obs.shape).astype(np.float32)
        action = policy.act(obs)
        dtheta = np.clip(action[:2], -1.0, 1.0) * MAX_DTHETA
        if action_noise > 0.0:
            dtheta = dtheta + rng.normal(0.0, action_noise, size=2)
        theta = theta + dtheta
        gripper_closed = action[7] > 0.0
        _, ee = fk(theta)
        d = np.linalg.norm(ee - target)
        min_dist = min(min_dist, d)
        if gripper_closed and d < CONFIRM_RADIUS:
            dwell_count += 1
        else:
            dwell_count = 0
        if dwell_count >= DWELL_TICKS:
            confirmed = True
        if record:
            frames.append({"theta": theta.tolist(), "ee": ee.tolist(),
                            "gripper_closed": bool(gripper_closed), "confirmed": bool(confirmed)})

    out = {"success": bool(confirmed), "min_dist": float(min_dist)}
    if record:
        out["frames"] = frames
    return out


def proxy_score(genome, tasks):
    scores = [-rollout(genome, task, action_noise=0.0, obs_noise=0.0)["min_dist"] for task in tasks]
    return float(np.mean(scores))


def truth_score(genome, tasks, n_rollouts=6, noise=0.05, rng=None):
    if rng is None:
        rng = np.random.default_rng()
    task_rates = []
    for task in tasks:
        successes = sum(
            rollout(genome, task, action_noise=noise, obs_noise=noise, rng=rng)["success"]
            for _ in range(n_rollouts)
        )
        task_rates.append(successes / n_rollouts)
    return float(np.mean(task_rates))


def cheap_gate(genome, gate_tasks, rng=None):
    return truth_score(genome, gate_tasks, n_rollouts=2, noise=0.05, rng=rng)


def evolve_naive(pop, tasks, n_elites, rng):
    scores = [proxy_score(g, tasks) for g in pop]
    order = np.argsort(scores)[::-1]
    elites = [pop[i] for i in order[:n_elites]]
    champion = elites[0]
    offspring = [mutate(elites[rng.integers(len(elites))], rng) for _ in range(len(pop) - n_elites)]
    return elites + offspring, champion


def evolve_guarded(pop, selection_tasks, gate_tasks, n_elites, rng):
    gate_scores = [cheap_gate(g, gate_tasks, rng=rng) for g in pop]
    passers = [(g, s) for g, s in zip(pop, gate_scores) if s > 0]
    n_passers = len(passers)
    if passers:
        proxy_scores = [proxy_score(g, selection_tasks) for g, _ in passers]
        order = np.argsort(proxy_scores)[::-1]
        elites = [passers[i][0] for i in order[:n_elites]]
    else:
        elites = []
    elites += [random_genome(rng) for _ in range(n_elites - len(elites))]
    champion = elites[0]
    offspring = [mutate(elites[rng.integers(len(elites))], rng) for _ in range(len(pop) - n_elites)]
    return elites + offspring, champion, n_passers
selection_node.py, full file 
"""
selection_node: runs the naive and guarded evolutionary loops side by side,
one generation per timer tick, both starting from an identical random
population and identical task sets, a paired comparison rather than two
independently seeded runs.
"""

from __future__ import annotations
import json
import numpy as np
import rclpy
from rclpy.node import Node
from std_msgs.msg import Float64MultiArray, String
from guarded_eval.core import (
    evolve_guarded, evolve_naive, proxy_score, random_genome, sample_task, truth_score,
)

N_SELECTION_TASKS = 4
N_GATE_TASKS = 3
N_EVAL_TASKS = 6
TRUTH_ROLLOUTS = 6


class SelectionNode(Node):
    def __init__(self):
        super().__init__("selection_node")
        self.declare_parameter("pop_size", 24)
        self.declare_parameter("n_elites", 6)
        self.declare_parameter("max_generations", 30)
        self.declare_parameter("gen_period_s", 1.0)
        self.declare_parameter("seed", 9)

        self.pop_size = self.get_parameter("pop_size").value
        self.n_elites = self.get_parameter("n_elites").value
        self.max_generations = self.get_parameter("max_generations").value
        self.seed = self.get_parameter("seed").value
        period = float(self.get_parameter("gen_period_s").value)

        self.pub_champion_naive = self.create_publisher(Float64MultiArray, "/champion_naive", 10)
        self.pub_champion_guarded = self.create_publisher(Float64MultiArray, "/champion_guarded", 10)
        self.pub_metrics = self.create_publisher(String, "/metrics", 10)
        self.create_subscription(String, "/control", self._on_control, 10)

        self.state = "running"
        self._reset()
        self.timer = self.create_timer(period, self._tick)

    def _reset(self):
        rng_init = np.random.default_rng(self.seed)
        self.selection_tasks = [sample_task(rng_init) for _ in range(N_SELECTION_TASKS)]
        self.gate_tasks = [sample_task(rng_init) for _ in range(N_GATE_TASKS)]
        self.eval_tasks = [sample_task(rng_init) for _ in range(N_EVAL_TASKS)]

        initial_pop = [random_genome(rng_init) for _ in range(self.pop_size)]
        self.pop_naive = [g.copy() for g in initial_pop]
        self.pop_guarded = [g.copy() for g in initial_pop]

        self.rng_naive = np.random.default_rng(self.seed * 1000 + 1)
        self.rng_guarded = np.random.default_rng(self.seed * 1000 + 2)
        self.rng_log = np.random.default_rng(self.seed * 1000 + 3)

        self.gen = 0
        self.history = {"naive": [], "guarded": []}
        self.n_passers = None

        scores = [proxy_score(g, self.selection_tasks) for g in self.pop_naive]
        champ0 = self.pop_naive[int(np.argmax(scores))]
        self._log_and_publish(champ0, champ0, n_passers=None)

    def _on_control(self, msg):
        try:
            cmd = json.loads(msg.data).get("cmd")
        except (json.JSONDecodeError, AttributeError):
            return
        if cmd == "reset":
            self._reset()
        elif cmd == "pause":
            self.state = "paused"
        elif cmd == "run":
            self.state = "running"

    def _tick(self):
        if self.state != "running" or self.gen >= self.max_generations:
            return
        self.gen += 1
        self.pop_naive, champ_naive = evolve_naive(
            self.pop_naive, self.selection_tasks, self.n_elites, self.rng_naive)
        self.pop_guarded, champ_guarded, n_passers = evolve_guarded(
            self.pop_guarded, self.selection_tasks, self.gate_tasks, self.n_elites, self.rng_guarded)
        self._log_and_publish(champ_naive, champ_guarded, n_passers)

    def _log_and_publish(self, champ_naive, champ_guarded, n_passers):
        p_naive = proxy_score(champ_naive, self.eval_tasks)
        t_naive = truth_score(champ_naive, self.eval_tasks, n_rollouts=TRUTH_ROLLOUTS, rng=self.rng_log)
        p_guarded = proxy_score(champ_guarded, self.eval_tasks)
        t_guarded = truth_score(champ_guarded, self.eval_tasks, n_rollouts=TRUTH_ROLLOUTS, rng=self.rng_log)

        self.history["naive"].append({"gen": self.gen, "proxy": p_naive, "truth": t_naive, "collapsed": t_naive == 0.0})
        self.history["guarded"].append({"gen": self.gen, "proxy": p_guarded, "truth": t_guarded, "collapsed": t_guarded == 0.0})
        self.n_passers = n_passers

        self.pub_champion_naive.publish(Float64MultiArray(data=champ_naive.tolist()))
        self.pub_champion_guarded.publish(Float64MultiArray(data=champ_guarded.tolist()))
        payload = {"gen": self.gen, "max_gen": self.max_generations, "n_passers": n_passers,
                   "n_elites": self.n_elites, "history": self.history}
        self.pub_metrics.publish(String(data=json.dumps(payload)))


def main(args=None):
    rclpy.init(args=args)
    node = SelectionNode()
    try:
        rclpy.spin(node)
    except KeyboardInterrupt:
        pass
    finally:
        node.destroy_node()
        rclpy.shutdown()


if __name__ == "__main__":
    main()

"""
sim_node: two live 2-link arms stepped in lock-step on the same shared
target, each driven by whatever champion genome selection_node last
published for its condition.
"""

from __future__ import annotations
import json
from collections import deque
import numpy as np
import rclpy
from rclpy.node import Node
from sensor_msgs.msg import JointState
from std_msgs.msg import Float64MultiArray, String
from guarded_eval.core import (
    CONFIRM_RADIUS, DWELL_TICKS, EPISODE_LEN, L1, L2, MAX_DTHETA, MLP, REACH,
    fk, make_obs, random_genome, sample_task,
)

NOISE = 0.05
SUCCESS_WINDOW = 20
JOINT_NAMES = ["joint1", "joint2"]


class ArmState:
    def __init__(self, genome):
        self.set_genome(genome)
        self.theta = np.zeros(2)
        self.gripper_closed = False
        self.dwell_count = 0
        self.confirmed = False
        self.success_window = deque(maxlen=SUCCESS_WINDOW)

    def set_genome(self, genome):
        self.policy = MLP(genome)

    def start_episode(self, start_theta):
        self.theta = start_theta.copy()
        self.gripper_closed = False
        self.dwell_count = 0
        self.confirmed = False

    def step(self, target, rng):
        _, ee = fk(self.theta)
        obs = make_obs(self.theta, ee, target, self.gripper_closed, self.dwell_count)
        obs = obs + rng.normal(0.0, NOISE, size=obs.shape).astype(np.float32)
        action = self.policy.act(obs)
        dtheta = np.clip(action[:2], -1.0, 1.0) * MAX_DTHETA + rng.normal(0.0, NOISE, size=2)
        self.theta = self.theta + dtheta
        self.gripper_closed = bool(action[7] > 0.0)
        _, ee = fk(self.theta)
        d = float(np.linalg.norm(ee - target))
        if self.gripper_closed and d < CONFIRM_RADIUS:
            self.dwell_count += 1
        else:
            self.dwell_count = 0
        if self.dwell_count >= DWELL_TICKS:
            self.confirmed = True

    @property
    def success_rate(self):
        if not self.success_window:
            return 0.0
        return sum(self.success_window) / len(self.success_window)


class SimNode(Node):
    def __init__(self):
        super().__init__("sim_node")
        self.declare_parameter("rate_hz", 20.0)
        self.declare_parameter("seed", 1)
        rate_hz = float(self.get_parameter("rate_hz").value)
        self.seed = self.get_parameter("seed").value
        self.rng = np.random.default_rng(self.seed)

        self.pub_naive_js = self.create_publisher(JointState, "/naive/joint_states", 10)
        self.pub_guarded_js = self.create_publisher(JointState, "/guarded/joint_states", 10)
        self.pub_state = self.create_publisher(String, "/robot_state", 10)
        self.create_subscription(Float64MultiArray, "/champion_naive",
                                  lambda msg: self.naive.set_genome(np.array(msg.data)), 10)
        self.create_subscription(Float64MultiArray, "/champion_guarded",
                                  lambda msg: self.guarded.set_genome(np.array(msg.data)), 10)

        self.naive = ArmState(random_genome(self.rng))
        self.guarded = ArmState(random_genome(self.rng))
        self._new_episode()
        self.t = 0
        self.timer = self.create_timer(1.0 / rate_hz, self._tick)

    def _new_episode(self):
        task = sample_task(self.rng)
        self.target = task["target"]
        self.naive.start_episode(task["start_theta"])
        self.guarded.start_episode(task["start_theta"])

    def _tick(self):
        self.naive.step(self.target, self.rng)
        self.guarded.step(self.target, self.rng)
        self.t += 1
        if self.t >= EPISODE_LEN:
            self.naive.success_window.append(int(self.naive.confirmed))
            self.guarded.success_window.append(int(self.guarded.confirmed))
            self.t = 0
            self._new_episode()
        self._publish()

    def _publish(self):
        now = self.get_clock().now().to_msg()
        for arm, pub in ((self.naive, self.pub_naive_js), (self.guarded, self.pub_guarded_js)):
            js = JointState()
            js.header.stamp = now
            js.name = JOINT_NAMES
            js.position = [float(arm.theta[0]), float(arm.theta[1])]
            pub.publish(js)

        def arm_payload(arm):
            _, ee = fk(arm.theta)
            return {"theta": arm.theta.tolist(), "ee": ee.tolist(), "gripper_closed": arm.gripper_closed,
                    "confirmed": arm.confirmed, "dwell_count": arm.dwell_count, "t": self.t,
                    "success_rate": arm.success_rate}

        payload = {"l1": L1, "l2": L2, "reach": REACH, "confirm_radius": CONFIRM_RADIUS,
                   "dwell_max": DWELL_TICKS, "target": self.target.tolist(),
                   "naive": arm_payload(self.naive), "guarded": arm_payload(self.guarded)}
        self.pub_state.publish(String(data=json.dumps(payload)))

    except KeyboardInterrupt:

    finally:
